🎯 Objetivo do Software

O Single Broker é um sistema de gestão para corretagem/imobiliária, com foco em organizar e centralizar o controle de imóveis, clientes e atendimentos.

O objetivo principal é facilitar o gerenciamento de processos internos para corretores, permitindo maior controle sobre cadastros, consultas e operações do dia a dia.

⚙️ Funcionalidades do Sistema
🏠 Gestão de Imóveis
Cadastro de imóveis
Edição de imóveis
Consulta de imóveis
Exclusão de registros
Associação de informações detalhadas do imóvel
👤 Gestão de Clientes
Cadastro de clientes
Atualização de dados
Consulta de clientes
Vinculação com imóveis e atendimentos
📞 Gestão de Contatos / Atendimentos
Registro de atendimentos
Histórico de contatos com clientes
Visualização de interações
🔐 Sistema de Usuários
Login de usuários
Controle de acesso por perfil
Permissões administrativas
🗃️ Banco de Dados
Persistência de dados via JPA/Hibernate
Integração com MySQL
